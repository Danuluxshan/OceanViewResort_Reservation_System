Add the following JARs to this folder:

1. MySQL Connector/J
   Download: https://dev.mysql.com/downloads/connector/j/
   Or: https://repo1.maven.org/maven2/com/mysql/mysql-connector-j/8.2.0/mysql-connector-j-8.2.0.jar
   Save as: mysql-connector-j-8.2.0.jar (or similar version)

2. Jakarta Mail (optional, for email notifications)
   jakarta.mail-api and angus-mail from https://eclipse-ee4j.github.io/mail/
